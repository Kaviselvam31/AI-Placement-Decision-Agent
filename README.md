# 🚀 AI Placement Decision Agent

An AI-powered career guidance platform that helps students evaluate their placement readiness, identify skill gaps, and receive a personalized roadmap for achieving their dream job.

The application analyzes a student's profile, resume, skills, interests, and career goals using Google's Gemini AI to generate intelligent recommendations, interview preparation guidance, and learning plans.

---

## 📌 Problem Statement

Many students struggle to determine whether they are ready for campus placements. They often don't know:

- Which companies they are eligible for
- Which skills they are missing
- What to study next
- How to prepare for interviews
- Which career path best matches their profile

This project solves these problems using Artificial Intelligence.

---

# 🤖 AI Agent Overview

The AI Placement Decision Agent acts as an intelligent virtual placement mentor.

It performs the following tasks:

- Reads student profile
- Analyzes uploaded resume
- Identifies strengths
- Detects missing skills
- Recommends suitable companies
- Generates interview questions
- Creates a personalized learning roadmap
- Provides placement guidance

Instead of giving fixed answers, the AI makes decisions based on the student's profile.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/Kaviselvam31/AI-Placement-Decision-Agent.git
```

Move into the project

```bash
cd AI-Placement-Decision-Agent
```

Install dependencies

```bash
npm install
```

Start the project

```bash
npm run dev
```

Open

```
http://localhost:5173
```

---

# 🔑 Environment Variables

Create a `.env` file

```
VITE_GEMINI_API_KEY=YOUR_API_KEY
```

Never upload your API key to GitHub.
